##  Deskripsi Proyek
Aplikasi **Dasbor Cuaca** ini menampilkan **cuaca saat ini** dan **prakiraan 5 hari ke depan** berdasarkan kota yang dicari oleh pengguna.  
Aplikasi ini memanfaatkan API dari [OpenWeatherMap](https://openweathermap.org/api) untuk menampilkan data cuaca real-time.

### Fitur Utama
- Pencarian kota dengan saran otomatis (autocomplete)
- Cuaca saat ini (suhu, kelembapan, kecepatan angin, ikon cuaca)
- Tabel prakiraan 5 hari ke depan
- Toggle satuan suhu °C / °F
- Riwayat pencarian kota (tersimpan di localStorage)
- Tampilan full-screen dengan tema biru dan CSS murni
- Responsif untuk tampilan desktop dan mobile
